 # Rules
- Do not open the server when you are done a task, unless you are asked to do so, or if you need it for something but in that case, you should close it when you are done.
- When implementing a change, always make sure to fully test it before you say that you are done, if tests including E2E or integrations are missing, add them.
- When creating orchestration examples, the default model should be claude-opus-4.5
